# Globo Video Downloader CLI
> Downloads videos from globo.com using your maximum network speed

![](http://g.recordit.co/maTNbLqbX8.gif)

## Requirements
- [axel](http://axel.alioth.debian.org/)

## Install
`$ npm install -g https://github.com/GuilhermeMedeiros/globo-downloader.git`

## Usage
`$ globo-downloader {link}`

## Note
- You need to have a subscription with globo.com
- You should not use this program to distribute videos
- Your credentials are saved without any encryption (you can delete them at ~/.globo_credentials)
